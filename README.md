# Machine Learning Based Ocean Wave Energy Forecasting

Machine learning-based ocean wave energy forecasting system using environmental parameters and predictive analytics.

## Overview

This project predicts Ocean Wave Energy using Machine Learning techniques. The system uses environmental parameters such as wave height, wave period, wind speed, and atmospheric conditions to forecast wave power density and improve renewable energy utilization.

## Objectives

* Analyze limitations of traditional wave forecasting methods.
* Apply Machine Learning models for wave energy prediction.
* Compare model performance using evaluation metrics.
* Improve forecasting accuracy for renewable energy planning.

## Technologies Used

* Python
* Django
* NumPy
* Pandas
* Scikit-learn
* SQLite

## Machine Learning Models

* Linear Regression
* Support Vector Regression (SVR)
* Random Forest
* XGBoost
* KNN

## Features

* Data Preprocessing
* Feature Selection
* Model Training and Evaluation
* Wave Energy Forecasting
* Performance Visualization

## Evaluation Metrics

* RMSE (Root Mean Square Error)
* MAE (Mean Absolute Error)
* R² Score
* MAPE (Mean Absolute Percentage Error)

## Results

The project demonstrates that Machine Learning models can improve ocean wave energy forecasting accuracy and support efficient renewable energy generation. Among the implemented models, XGBoost and Random Forest provided strong predictive performance for wave power estimation.
